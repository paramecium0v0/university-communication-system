# 如何运行项目

## 步骤 1: 打开终端
在 Mac 上，按 `Command + 空格键`，输入 "终端" 或 "Terminal"，然后按回车。

## 步骤 2: 进入项目目录
在终端中输入以下命令：

```bash
cd "/Users/llq/Desktop/vibe coding/2222"
```

## 步骤 3: 安装依赖
运行以下命令安装所需的包：

```bash
npm install
```

这可能需要几分钟时间，会下载所有必需的依赖项。

## 步骤 4: 启动开发服务器
安装完成后，运行：

```bash
npm run dev
```

## 步骤 5: 打开浏览器
终端会显示类似这样的信息：

```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

点击终端中的链接 `http://localhost:5173/`，或者手动在浏览器中打开这个地址。

## 完成！
现在你应该能看到大学通信系统的界面了！

---

## 如果遇到问题

### 如果 `npm install` 失败：
- 确保你已经安装了 Node.js（版本 16 或更高）
- 检查网络连接
- 尝试使用 `npm install --legacy-peer-deps`

### 如果端口被占用：
- Vite 会自动尝试其他端口（如 5174）
- 查看终端输出中的实际端口号

### 停止服务器：
- 在终端中按 `Ctrl + C` 停止开发服务器
